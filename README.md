## Allure Reports

This project uses Allure to generate test reports.

### Generating the report

To generate and view the Allure report, follow these steps:

1. Install dependences:
    ```bash
    npm i
    ```

1.  Run Cypress tests:
    ```bash
    npm run test
    ```


![Tests](image.png)


2.  Command to open Allure report:
    ```bash
    npm run allure
    ```
